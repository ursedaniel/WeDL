Introducere

Să se implementeze o extensie pentru navigator sau o componentă Web capabilă să interpreteze şi să ofere reprezentări structurate – ce pot fi exportate sub formă de HTML şi JSON incluzând scheme de microdate HTML5 – a unor (tipuri de) date incluse într-o pagină Web: numere de telefon, adrese poştale şi de e-mail, unităţi de timp, nume/categorii de resurse (imagini, melodii, filme), adrese Web (URL-uri date ca text sau drept cod QR), profil de utilizator (e.g., având cont public oferit de aplicaţii sociale ca Facebook, Reddit, Twitter etc.), coordonate geografice şi altele. Viitoare detectoare de formate vor putea fi incluse via extensii – se vor oferi, pentru început, minim 3.

Interfata

Designul se bazeaza pe principiile Material, folosind 2 culori principale (verde avand hexacodul #37a36c si rosu avand hexacodul #9e2a2a)

De asemenea se folosesc nonculori si derivate a celor 2 culori principale specificate mai sus.

Structura

WeDI este o extensie ce isi propune sa colecteze date intr-o maniera foarte usor de implementat.

Fisiere HTML

Fisierele html trebuie encodate cu UTF-8, trebuie sa contina titlu si metadate. De asemenea se cere o indentare cat mai precisa. Fisierele html nu au voie sa contina comentarii si trebui sa respecte in totalitate regulile de la W3.

CSS Files

Fisierele css trebuie sa fie cat mai structurate si compacte, se incearca sa se scrie cat mai putin cod si sa fie utilizat in cat mai multe locuri.

Fisierele JavaScript

Fisierele Javascript trebuie introduse la sfarsit de document. Ele vor fi perfect indentate, codul fiind scrie in cat mai multe functii unitare.

Wireframe-uri

Chrome API

Pentru extensia Chrome am folosit API-ul de la Google Chrome (storage, content-script, active-tab).

Plugin-urile pentru extensie sunt reprezentate de JSONS, avand urmatorul format:
{ "icon": "example", "name": "Value", "key": "Unique value", "algorithm": "function(param) {...}" }

Icon = cheia unica pentru iconita din Font-Awesome
Name = numele plugin-ului
Key = cheia unica de identificare a plugin-ului
Algorithm = un string reprezentat de cod valid Javascript, avand urmatoarea forma:
(function() {
return function (param) {
... parse data
... return data
};
}());
