Import

To import, you must have a .json file containing:

{ "icon": "example", "name": "Value", "key": "Unique value", "algorithm": "function(param) {...}" }

Icon: FontAwesome icon (eg: fa-cog)

Name: Name of plugin (eg: Email)

key: Unique value of plugin (eg: email)

Algorithms: A string which represent a javascript
function which take 1 parameter and return a json
The param will have the next form: {url: string, text: string}
